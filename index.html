SimpleBus/JMSSerializerBundleBridge

By Matthias Noback

This library contains a Symfony bundle which configures the ObjectSerializer from SimpleBus/JMSSerializerBridge as the default object serializer for SimpleBus/AsynchronousBundle. You only need to enable the SimpleBusJMSSerializerBundleBridgeBundle in your AppKernel to accomplish this.
